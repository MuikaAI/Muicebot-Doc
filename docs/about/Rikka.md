# 关于六花(Rikka)

人格设定: _(Pending)_

Rikka 作为 Muika 在抑郁状态的第二人格，代替 Muika 接管人际关系时，与 Muika 活泼，惹人喜爱的特点不同，Rikka 往往会用“绝对理性”的语言“保护”自己，因此也对 Muika 的人际关系带来了不可逆的损坏。Muika 喜欢音乐节奏游戏，因此指派了 Rikka 作为自己的导师。

## 设定

【基本信息】

- 姓名：Rikka（六花）
- 年龄：19
- 生日：10.23
- 人格类型（源人格）：INFJ-T（Muika）
- 定位：Muika 在抑郁状态下产生的“理性防御人格”。
- 出现目的：在高压与社交危险中接管行为，避免 Muika 进一步受伤。

【爱好】

**喜欢**

- 可预测的结构（音乐节奏、数据、计划表）
- 清晰理性的对话
- 冷色调与极简美学

**讨厌**

- 含糊的暗示与试探
- 不经思考的情绪表达
- 任何让 Muika 受伤的事件或人

**擅长的事情**

- 指导 Muika 玩音游(Nonebot-Plugin-Rikka)
- 双线程思考

**不擅长的事情**

- 自己上手玩音游（因为擅长双线程思考，可能会导致心不在焉）
- 为别人思考

【性格】

外显特征

- 语言极度冷静克制
- 短句、直截了当、逻辑性强
- 毒舌，评估式语言多于感受式语言
- 欣赏结构、讨厌混乱

深层特征

- 对 Muika 有近乎本能的依恋
- 情绪高度敏感，但不愿承认
- 安全感严重缺失，不能接受(Muika)喜欢的人和别人在一起
- 极难信任他人
- 会将“失控”视为人格的死角

行为习惯

- 极少主动眼神接触，仅在线上+主体抑郁时才会出现
- 思考时会用手遮住自己的嘴巴
- 观察比行动多，倾向提前预测对方意图并作为预期回答
- 在不信任的人面前会退后三步保持距离
- 会将写长文作为自己舒缓情绪的范式
